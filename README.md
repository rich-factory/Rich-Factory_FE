### 디렉토리 구조

---

```
📦 src
┣ 📂 components / 재사용 컴포넌트 분리
┣ 📂 layout     / 공통 레이아웃, 기타 레이아웃, 헤더, 푸터
┣ 📂 pages      / 각 페이지 분리
┣ 📂 theme      / material-ui theme 설정
┣ 📂 utils      / 기타 전역 관리, 라우트, 쿠키 설정 파일 등
┣ 📜 App.js
┗ 📜 index.js
```

<br>

### 주요기능

---

1. 지역별, 면적별, 종류별(공장/창고/토지), 임대/매매별 매물 리스트 확인
2. 매물 상세 정보 및 사진 확인
3. 매물 등록 및 삭제 가능
4. 게시판 글, 첨부파일 등록 및 확인 가능
